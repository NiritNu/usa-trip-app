# USA Trip App – Public Safe Starter

This version is safe to upload to a public GitHub repository.

It does **not** include:
- Real flight details
- Real hotel names or check-in dates
- Family names or children details
- Ticket windows
- Reservation numbers
- Private itinerary dates

## Run locally

```bash
npm install
npm run dev
```

## Public GitHub Pages

This project is designed to be published as a simple static website.

Keep private travel details out of GitHub if the repository is public.
Use placeholders in `src/data/trip.json`, or later connect a private backend such as Firebase/Supabase.
